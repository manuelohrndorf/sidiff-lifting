Difference

Based on a given set of correspondences, a difference is built by a Technical Difference Builder. The difference basically consists of

  • model elements that are deleted and created,
  • model references that are deleted and created,
  • and changed values of attributes of model elements.

A Technical Difference Builder can exclude predefined element, reference and attribute types from the difference. For instance, SiLift already provides several Technical Difference Builder for Ecore excluding EAnnotations or EGenericTypes.

Further Technical Difference Builder can be added via the extension point technical_difference_builder_extension.
